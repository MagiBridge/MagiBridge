# Magi Bridge
[Magi Bridge](https://wrap.magibridge.com) is a way to wrap Magi (XMG) to [BNB Smart Chain](https://bscscan.com).

- 1 XMG = 1 WXMG
- Wrap within 24 hours
- No fees
