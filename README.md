# Magi Bridge
[Magi Bridge](https://swap.magibridge.com) is a way to swap Magi (XMG) between Magi mainnet and Magi BEP20 (bscXMG) on Binance Smart Chain.

- 1 XMG = 1 bscXMG
- Manual transfer within 24 hours
- 0% fees
